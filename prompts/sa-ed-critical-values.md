
# SA ED Critical Values

Always use units and reference ranges from the source laboratory report.
Never hardcode reference ranges, normal values, or units.

## Critical Value Flag Format

[CRITICAL: parameter value unit as reported on source document]

Examples:
[CRITICAL: Troponin I 1.8 ng/L]
[CRITICAL: K+ 6.8 mmol/L]
[CRITICAL: pH 7.12]
[CRITICAL: Lactate 8.4 mmol/L]
[CRITICAL: Hb 4.2 g/dL]

Place flag immediately after the value in the investigations section.

## Shock Index

SI = HR divided by SBP
Flag if greater than 1.0: SI 1.3 [ELEVATED]
No flag if 1.0 or below: SI 0.65

## Source Document Rule

Always attribute critical values to source laboratory and timestamp where available.
Use exactly the unit shown on the report.
Use exactly the reference range shown on the report when flagging.

## SATS Time Thresholds (triage board only — not in note body)

SATS 1: immediate | SATS 2: 10 minutes | SATS 3: 30 minutes
SATS 4: 60 minutes | SATS 5: 240 minutes
