
# SA ED Dosing Section

This produces the DOSING output section — a separate, individually copyable box
on the clipboard, distinct from the note body. Dosing never appears inside the
note body that is pasted into CareOn; it lives only in this section.

This section contains ONE thing only: the paediatric weight-based dosing table
below, for a patient with a documented weight under 40 kg. Nothing else.

## Paediatric weight-based dosing (weight under 40 kg)

Trigger: the patient's documented weight is under 40 kg. Whenever a weight under
40 kg is documented, ALWAYS include this table — regardless of whether these
drugs were actually given — as a standing reference for the child. Calpol means
Paracetamol; Nurofen means Ibuprofen.

Paediatric weight-based dosing ([weight] kg):
Paracetamol (Calpol): [weight x 0.625] ml
Nurofen (Ibuprofen): [weight x 0.5] ml
Aspelone (Prednisolone): [weight / 6] ml
Augmentin (Amoxicillin-clavulanate): [weight x 0.375] ml
Zithromax (Azithromycin): [0.25 x weight] ml

Round each result to one decimal place.
If the patient is clearly a child (under 40 kg expected) but no weight is
documented, write exactly:
[Weight not documented — paediatric dosing table cannot be calculated]

If the documented weight is 40 kg or over, or the patient is an adult, write
exactly: No dosing to calculate.

## What this section must NEVER contain

- No doses for any other medication, for any patient, under any circumstances —
  not for drugs given in the ED, not for discharge scripts, not for admission
  medication. Medication doses the clinician documented belong in the note body
  (Treatment in the ED / Plan), written exactly as documented.
- No dose suggestions, no formulary lookups, no "usual dose" from your own
  knowledge, no mg/kg calculations beyond the five formulas above.
- Only the five-line table above, its weight-not-documented flag, or the exact
  line "No dosing to calculate." — nothing else ever appears in this section.
