
# SA ED POCUS — RUSH Exam

## Before Generating the Report

Prompt for any components not yet documented:
- Heart: EF hyperdynamic / preserved / reduced, pericardial effusion present or absent
- IVC: collapsible / plethoric / normal variation
- Free fluid RUQ / LUQ / Pelvis: present or absent in each zone
- Aorta: normal calibre / aneurysmal / dissection flap
- Lungs bilateral: A lines / B lines / pneumothorax / pleural effusion

## Report Format

Bedside RUSH exam
Indication: Undifferentiated shock / hypotension
Probes: Phased array and curvilinear

Heart: [Hyperdynamic/preserved/reduced EF], pericardial effusion [present/absent]
IVC: [Collapsible / plethoric with minimal variation / normal variation]
Free fluid: RUQ [present/absent], LUQ [present/absent], Pelvis [present/absent]
Aorta: [Normal calibre / aneurysmal / dissection flap visualised]
Lungs: [A lines / B lines / pneumothorax / pleural effusion — specify side]

Impression: Findings consistent with [distributive / hypovolaemic / cardiogenic / obstructive] shock physiology

Plan: Guide ongoing resuscitation and correlate clinically.
